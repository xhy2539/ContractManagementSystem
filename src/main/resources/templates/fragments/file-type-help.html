支持的文件类型说明
文档类型
  • PDF便携式文档格式
  • DOC/DOCXMicrosoft Word 文档
  • XLS/XLSXMicrosoft Excel 表格
  • PPT/PPTXMicrosoft PowerPoint 演示文稿
  • TXT纯文本文件
  • RTF富文本格式
图片类型
  • JPG/JPEG常用照片格式
  • PNG透明背景图片
  • GIF动画图片
  • BMP位图格式
  • WEBP现代Web图片格式
  • SVG矢量图形
  • TIFF高质量图片
压缩文件
  • ZIP通用压缩格式
  • RARWinRAR压缩格式
  • 7Z7-Zip压缩格式
  • TARUnix归档格式
  • GZIPGNU压缩格式
多媒体文件
  • MP4视频文件
  • AVI视频文件
  • MP3音频文件
  • WAV无损音频
数据文件
  • JSON数据交换格式
  • XML标记语言
  • CSV逗号分隔值文件
文件大小限制：
  • 普通文件：最大 10MB
  • 压缩包/多媒体文件：最大 50MB
支持功能：
  • ✅ 断点续传：网络中断后可继续上传
  • ✅ 自动重试：失败后智能重试机制
  • ✅ 在线预览：PDF、图片、文本、多媒体文件
  • ✅ 安全下载：所有文件类型均可下载
